# MatchThree #

Programming assignment for course DAT055, Chalmers University of Technology.

## Directory structure ##

| Directory         | Purpose         |
| :---------------- | :-------------- |
| `resource/`       | Run-time assets |
| `src/`            | Source code     |
| `src/Controller/` | Controllers     |
| `src/GameModes/`  | Game modes      |
| `src/Model/`      | Models          |
| `src/View/`       | Views           |
| `target/`         | Built artifacts |
